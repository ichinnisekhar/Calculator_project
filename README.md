# Python Calculator

This is a simple command-line calculator built using Python.I created this project while learning Python to practice concepts like loops, conditional statements, user input, and basic arithmetic operations.

# Features
* Addition of multiple numbers
* Subtraction of multiple numbers
* Multiplication of multiple numbers
* Division of two numbers
* Handles division by zero
* Menu-driven interface
* Exit option to close the program

# Technologies Used
* Python 3
